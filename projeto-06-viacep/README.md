# 🌐 Buscador de Endereço por CEP

Ferramenta desenvolvida em Python para buscar endereços completos
automaticamente através do CEP, utilizando a API pública do ViaCEP.

## 💡 Contexto
Desenvolvido por um corretor de imóveis em transição para a área
de tecnologia. Ferramenta útil para preencher endereços de imóveis
automaticamente durante o cadastro, evitando erros de digitação.

## 🚀 Como usar
1. Execute o arquivo `buscador.py`
2. Digite o CEP (com ou sem traço)
3. O programa busca e exibe o endereço completo automaticamente

## 📊 Exemplo de resultado
```
===== ENDEREÇO ENCONTRADO =====
CEP:          03530-010
Logradouro:   Rua Plínio de Carvalho
Complemento:  
Bairro:       Vila Matilde
Cidade:       São Paulo
Estado:       SP
```

## 🔗 API utilizada
[ViaCEP](https://viacep.com.br) — API pública e gratuita de consulta de CEPs brasileiros.

## 🛠️ Tecnologias
- Python 3.13
- Biblioteca `requests`
