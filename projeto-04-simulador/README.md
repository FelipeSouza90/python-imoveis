# 🤝 Simulador de Proposta e Contraproposta

Ferramenta desenvolvida em Python para simular negociações
imobiliárias, sugerindo contrapropostas intermediárias entre
vendedor e comprador.

## 💡 Contexto
Desenvolvido por um corretor de imóveis em transição para a área
de tecnologia. Ferramenta útil para conduzir negociações de forma
mais objetiva e embasada.

## 🚀 Como usar
1. Execute o arquivo `simulador.py`
2. Informe o valor pedido pelo vendedor
3. Informe o valor ofertado pelo comprador
4. O programa sugere 3 opções de negociação

## 📊 Exemplo de resultado
```
===== RELATÓRIO DE NEGOCIAÇÃO =====
Valor pedido pelo vendedor:    R$ 850,000.00
Valor ofertado pelo comprador: R$ 750,000.00
Diferença:                     R$ 100,000.00 (11.8%)

--- SUGESTÕES DE NEGOCIAÇÃO ---
Meio termo:                    R$ 800,000.00
Vendedor cede 3%:              R$ 824,500.00
Comprador sobe 3%:             R$ 772,500.00
```
