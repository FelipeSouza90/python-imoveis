# 🏠 Python no Mercado Imobiliário

Repositório com projetos Python desenvolvidos por um corretor de imóveis 
em transição para a área de tecnologia.

A ideia é unir o conhecimento do mercado imobiliário com programação,
criando ferramentas úteis para o dia a dia do corretor.

## 📁 Projetos

## 📁 Projetos

| Projeto | Descrição | Conceitos |
|---------|-----------|-----------|
| [01 - Calculadora de Financiamento](./projeto-01-calculadora) | Simula parcelas, total pago e juros de um financiamento | Variáveis, input, formatação |
| [02 - Analisador de Perfil do Comprador](./projeto-02-analisador) | Verifica se o cliente tem renda suficiente para financiar | if/else, operadores |
| [03 - Organizador de Carteira de Imóveis](./projeto-03-carteira) | Cadastra imóveis e calcula o valor total da carteira | Listas, dicionários, loops |
| [04 - Simulador de Proposta e Contraproposta](./projeto-04-simulador) | Simula negociações sugerindo contrapropostas | Funções, return |
| [05 - Gerador de Relatório de Visitas](./projeto-05-relatorio) | Registra visitas do dia e salva em arquivo .txt | Arquivos, datetime |
| [06 - Buscador de Endereço por CEP](./projeto-06-viacep) | Busca endereço completo automaticamente pelo CEP | APIs, requests |
| [07 - Sistema de Gestão de Leads](./projeto-07-sistema) | Sistema completo de gestão de leads imobiliários | Tudo integrado |


## 🛠️ Tecnologias
- Python 3.13
- Biblioteca `requests`
- Biblioteca `datetime`
- API ViaCEP
