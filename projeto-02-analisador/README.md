# 🏠 Analisador de Perfil do Comprador

Ferramenta desenvolvida em Python para analisar se um cliente
tem perfil aprovado para financiamento imobiliário, baseado
na regra dos bancos: a parcela não pode ultrapassar 30% da renda mensal.

## 💡 Contexto
Desenvolvido por um corretor de imóveis em transição para a área 
de tecnologia. Ferramenta útil para pré-qualificar clientes antes 
de apresentar imóveis, evitando visitas desnecessárias.

## 🚀 Como usar
1. Execute o arquivo `analisador.py`
2. Preencha os dados solicitados:
   - Renda mensal do cliente
   - Valor do imóvel
   - Valor da entrada
   - Prazo em anos
   - Taxa de juros anual

## 📊 Exemplo de resultado
```
❌ CLIENTE REPROVADO!
Parcela:          R$ 3,799.65
Limite (30%):     R$ 3,150.00
Renda mensal:     R$ 10,500.00
Renda necessária: R$ 12,665.51
```
