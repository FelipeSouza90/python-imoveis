# 📋 Organizador de Carteira de Imóveis

Ferramenta desenvolvida em Python para cadastrar e organizar
imóveis da carteira do corretor, exibindo um relatório completo
com valor total da carteira ao final.

## 💡 Contexto
Desenvolvido por um corretor de imóveis em transição para a área
de tecnologia. Ferramenta útil para organizar e ter uma visão
rápida da carteira de imóveis disponíveis.

## 🚀 Como usar
1. Execute o arquivo `carteira.py`
2. Informe quantos imóveis deseja cadastrar
3. Preencha os dados de cada imóvel:
   - Endereço
   - Tipo (apto/casa/studio)
   - Valor

## 📊 Exemplo de resultado
```
===== CARTEIRA DE IMÓVEIS =====
Total de imóveis cadastrados: 3

Imóvel 1
  Endereço: Av. Pássaros
  Tipo:     Casa
  Valor:    R$ 850,000.00

Imóvel 2
  Endereço: Av. Canários
  Tipo:     Apto
  Valor:    R$ 845,600.00

Valor total da carteira: R$ 2,255,600.00
```
