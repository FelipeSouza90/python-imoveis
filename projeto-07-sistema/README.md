# 🏆 Sistema de Gestão de Leads Imobiliários

Sistema completo desenvolvido em Python para cadastrar, analisar
e gerenciar leads imobiliários, unindo todos os conceitos aprendidos
na trilha Python no Mercado Imobiliário.

## 💡 Contexto
Projeto final desenvolvido por um corretor de imóveis em transição
para a área de tecnologia. Reúne ferramentas práticas do dia a dia
do corretor em um único sistema integrado.

## ✨ Funcionalidades
- Cadastrar leads com dados completos
- Buscar endereço automaticamente pelo CEP
- Analisar perfil financeiro do cliente
- Listar todos os leads cadastrados
- Gerar relatório completo em arquivo .txt

## 🚀 Como usar
1. Execute o arquivo `sistema.py`
2. Navegue pelo menu:
   - Opção 1: Cadastrar novo lead
   - Opção 2: Listar todos os leads
   - Opção 3: Salvar relatório em .txt
   - Opção 4: Sair
