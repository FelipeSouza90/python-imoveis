# 📄 Gerador de Relatório de Visitas

Ferramenta desenvolvida em Python para registrar visitas realizadas
pelo corretor durante o dia, salvando automaticamente um relatório
completo em arquivo .txt.

## 💡 Contexto
Desenvolvido por um corretor de imóveis em transição para a área
de tecnologia. Ferramenta útil para documentar visitas, registrar
o nível de interesse dos clientes e gerar relatórios organizados
para acompanhamento.

## 🚀 Como usar
1. Execute o arquivo `relatorio.py`
2. Informe seu nome e quantidade de visitas realizadas
3. Preencha os dados de cada visita:
   - Endereço do imóvel
   - Nome do cliente
   - Nível de interesse (alto/médio/baixo)
   - Observações
4. O relatório é salvo automaticamente em `.txt`

## 📊 Exemplo de resultado
```
==================================================
RELATÓRIO DE VISITAS
Data: 28/03/2026 | Hora: 10:44
Corretor: Felipe
Total de visitas: 3
==================================================

Visita 1
  Endereço:   Rua Estevão Lopes, 136
  Cliente:    Fábio
  Interesse:  Médio
  Observação: Precisa verificar questão de entrada

==================================================
Relatório gerado automaticamente pelo sistema.
```
